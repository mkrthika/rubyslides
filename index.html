Ruby Variables

Variables are the memory locations, which hold any data to be used by any program.There are four types of variables supported by Ruby.

  1. Global Variables
  2. Instance Variables
  3. Class Variables
  4. Local Variables

Global Variables

Global variables begin with $.Global Variables can be accessed by more than 2 classes.

Instance Variables

Instance variables begin with @. Uninitialized instance variables have the value nil and produce warnings with the -w option.

Class Variables

Class variables begin with @@ and must be initialized before they can be used in method definitions.Class variables are shared among descendants of the class or module in which the class variables are defined.

Local Variables

Local variables are local to the code construct in which they are declared. For example, a local variable declared in a method or within a loop cannot be accessed outside of that loop or method. Local variable names must begin with either an underscore or a lower case letter.
